ANOTAÇÕES WORDPRESS

1-Estrutura do worpress

-  o arquivo wp-config é dos mais importante, porque nele vai ter configurações de banco de dados, de usarios ...

- a pasta wp-admin é responsável por todas configurações do painel administrativo

- Na pasta wp-content está as subpastas com plugis, temas ...

2-ESTRUTURA DE UM TEMA

- DENTRO DO TEMA UTILIZADO TEM O ARQUIVO INDEX.PHP E DENTRO DELE TEM AS FUNCÕES QUE VÃO SER RESPONSÁVEIS POR CHAMAR CADA PARTE DA PAGINA INDEX

EXEMPLO: A FUNÇÃO get_header(); É RESPONSAVEL PELO CABECALHO DENTRE OUTRAS FUNÇÕES.

3 -CRIANDO UM TEMA
 3.1 PARA CRIAR UM TEMA, DEVE-SE CRIAR UMA SUBPASTA NA PASTA TEMA COM O NOME DO TEMA (COM LETRAS MINUSCULAS)
 3.2 CRIAR UM ARQUIVO INDEX.PHP DENTRO DA SUBPASTA E CRIAR UM ARQUIVO STYLE. ESSES ARQUIVOS SÃO NECESSARIO PARA O WORDPRESS VALIDAR O TEMA. 
 NO ARQUIVO STYLE DECLARA-SE ALGUMAS INFORMAÇÕES IMPORTANTE, COMO: NOME DO TEMA, 
 AUTHOR ...POREM PARA SER CONSIDERADO UM TEMA WORDPRESS PRECISA SABER NO MININO O NOME DO TEMA, AS OUTRAS INFORMAÇÕES SÃO OPCIONAIS. LOGO DEPOIS, ATIVAR O TEMA CRIADO


 4 - FUNCTIONS E HOOKS(HOOKS significa gancho)
  4.1  DENTRO DO TEMA TEM UM ARQUIVO CHAMADO FUNCTIONS.PHP NELE VAI TODAS AS FUNÇÕES REPONSAVEIS POR IMPLEMENTAR AS LOGICAS DAS PAGINA QD TIVER.(essa funçao é carregada até ante do tema)

  4.2 COM O HOOKS PODE-SE PROGRAMAR DETEMINADAS AÇÕES (ou seja chamar deteminada funçao em algum momento...)


5 - UMA BOA PRATICA PARA USAR ARQUIVOS CSS E JS É ATRAVEŚ DE HOOKS


2 - Informações impoprtantes

- LEMBRAR QUE AO ENTRAR NA PAGINA INICIAL DO WORDDPRESS JA VAI TER UMA PAGINA DEFINIDA COM O NOME QUE FOI DADO NAS CONFIGURAÇÕES INICIAIS, TAMBÉM VAI HAVER UM POST E UM COMENTARIO QUE SÃO DEFINIDOS COMO PADRÃO DA FERRAMENTA. NO ENTANTO, O USUARIO TEM TOTAL LIBERDADE PARA INSERIR MAIS PAGINA, POST .... 

- PARA CHAMAR AS PASTAS PADROES DO WORDPRESS USAR O VOLUME DETERMINADO NO SERVIÇO DO WORDPRESS.

- PARA INSERIR CONTEUDO EM UMA PAGINA ESPECIFICA, CLICAR NO LINK EDITAR DA PAGINA DESEJADA

- PARA TROCAR DE TEMA, BASTA PROCURAR POR UM TEMA NA ABA APARENCIA E ESCOLHER UM TEMA DESEJADO.

- PARA INSERIR UM PLUGUIN, BASTA PROCURAR POR PLUGINS NA ABA PLUGIN

- PARA DESABILITAR A BARRA DE FERRAMENTA, ENTRAR NO MENU PERFIL E DESMARACAR A OPÇÃO
